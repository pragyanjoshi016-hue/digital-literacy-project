## 📖 Project Overview

This repository is my **Digital Literacy Portfolio**, created as part of the CSE0001 course at VIT Bhopal University.

In this project, I worked as a **Student Digital Ambassador**, focusing on helping students understand digital tools, online safety, and professional communication.

The portfolio includes **five tasks**, each linked to a course module:

* Digital awareness creation
* Professional online presence
* Coding and collaboration tools
* Email communication
* Cybercrime awareness
